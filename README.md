# 📊 India State Debt Dashboard (2018-2026)

This project is an **interactive R Shiny application** designed to visualize and compare 
the fiscal health of Indian states using the **Debt-to-GSDP ratio**.

## 🎯 Key Insights
- Identifies states exceeding the **30% sustainability threshold**.
- Compares **BIMARU** states vs. **High Income** states.
- Analyzes the fiscal impact of the 2020-21 economic shock.
